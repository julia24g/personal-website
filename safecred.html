Helping Hands

Top Three Hacks

The unfortunate reality of donating to charity is that you don’t know where your money is being spent or if it’s going towards the cause at all.

Helping Hands was designed to let donators track exactly where their dollars go.

Technical

Helping Hands is a beta web application built using Hedera Hashgraph, a distributed ledger technology that has its own cryptocurrency called hbar.

To create our application, we first created a Gradle project in Java. We then created two Hedera testnet accounts to mimic the interaction between a donator and a charitable organization; each account started with 1000 hbar.

Once we developed the functionality to transfer hbar on the distributed ledger, we needed to build the front-end using HTML and CSS and connect the back-end using a web framework. In this case, using Spring-boot would have worked well - however, no team members knew Spring well enough to implement it in the remainder of time we had left.

After trying to work through our greatest challenge with this project, we discovered a quick and simple way of connecting the front-end and back-end pieces by using a Java library on Stack Overflow called ‘Render Snake’.
